# Quick installation script

## For Ubuntu-based systems

```bash
git clone https://github.com/truong51972/scripts.git ~/.quicksetup
cd ~/.quicksetup
bash setup-ubuntu.sh
cd ..
rm -rf ~/.quicksetup
```

For best experience with powerlevel10k, it is recommended to use a terminal that supports Nerd Fonts, such as Windows Terminal, iTerm2, or Alacritty. You can install a Nerd Font from [Nerd Fonts](https://www.nerdfonts.com/) or quick [link](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/Meslo.zip)
